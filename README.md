\# P2P Messenger



P2P мессенджер без открытых портов, работающий через WebRTC.



\## Особенности



\- 🔒 Прямое P2P соединение между пользователями

\- 🚫 Не требует открытых портов (использует WebRTC)

\- 💬 Мгновенные сообщения

\- 👥 Список онлайн пользователей

\- 📱 Адаптивный дизайн (работает на ПК и Android)



\## Как это работает



1\. \*\*Signaling сервер\*\* (только для установки соединения)

&#x20;  - Помогает пользователям найти друг друга

&#x20;  - Передает WebRTC offer/answer и ICE кандидаты

&#x20;  - Не хранит сообщения



2\. \*\*WebRTC соединение\*\*

&#x20;  - Прямое P2P соединение после установки

&#x20;  - Использует STUN серверы для обхода NAT

&#x20;  - DataChannel для передачи сообщений
